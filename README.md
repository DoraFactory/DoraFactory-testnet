# dora-vota
doravota node which supports cosmwasm in cosmos ecosystem

## Install and Using
> requirement: go version 1.20+

1. download
```shell
git clone https://github.com/DoraFactory/doravota.git
```

2. compile
```
cd doravota && make install
sudo cp ~/go/bin/dorad /usr/local/bin
```

3. create wallet
```
dorad keys add xxx
```
